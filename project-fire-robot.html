PROJECT DETAIL

화재감지 순찰 로봇

기간 · 2025.09.22 ~ 2025.11.27

기술 · ROS2 Humble, Navigation2, slam_toolbox, FastAPI, WebSocket, Raspberry Pi, Arduino(MQ-2, ENS160)

프로젝트 개요

실내 환경을 자율 순찰하며 화재·연기·유해가스를 감지하고, 웹 대시보드와 연동해 상태를 모니터링하는 로봇 시스템입니다. SLAM으로 지도 생성 후 Navigation2로 경로 계획을 수행하고, PatrolManager 노드로 순찰 시나리오를 자동화했습니다.

내 역할

  • · SLAM 지도 생성 및 Nav2 파이프라인 환경 구성
  • · YAML 기반 waypoints/zone 설계 및 PatrolManager.py 구현
  • · start/pause/resume/return 명령 처리, 배터리 모니터링 및 자동 복귀 로직 개발
  • · FastAPI 웹 대시보드와 ROS2 사이의 WebSocket 연동 구조 설계
