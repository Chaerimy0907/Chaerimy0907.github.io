PROJECT DETAIL
화재감지 순찰 로봇
ROS2 · SLAM · Navigation2 · Patrol System · FastAPI · WebSocket · Raspberry Pi · Arduino

프로젝트 개요

실내 환경을 자율 순찰하며 화재·연기·유해가스를 감지하고, 웹 대시보드에서 실시간으로 상태를 모니터링할 수 있는 로봇 시스템입니다.

내 역할

  • · slam_toolbox를 이용한 실내 지도 작성 및 Nav2에서 활용 가능하도록 맵 구성
  • · Navigation2 파이프라인(AMCL, planner, controller, costmap) 환경 구성 및 안정화
  • · YAML 기반 waypoints / zone 구조 설계 및 PatrolManager.py 구현
  • · start / pause / resume / return / repeat 명령 처리 로직 개발
  • · 배터리 상태 모니터링 및 임계치 도달 시 자동 복귀 로직 구현
  • · FastAPI 웹 대시보드와 ROS2 사이 WebSocket 연동 구조 설계
