PPE 착용 감지 실험

기술 · YOLOv8, Python

작업자의 안전모/조끼 착용 여부를 감지하고 일정 시간 단위로 착용률을 계산해 CSV로 저장하는 실험 프로젝트입니다. YOLOv8 모델 구조 이해와 후처리 로직 설계, 로그 저장 구조 설계에 집중했습니다.

  • · YOLOv8 기반 PPE Detection 파이프라인 구성
  • · 감지 결과로 착용률 계산 및 CSV 로깅 기능 구현
  • · 코드 리팩토링 및 모듈 분리 실습
